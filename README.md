# Barbería App

Una aplicación web sencilla con **FastAPI** que muestra los servicios y precios de una barbería, permite a los clientes solicitar citas y guarda la información en una base de datos SQLite. También incluye una tabla para visualizar todas las citas programadas.

## Características

- Página principal con lista de servicios y precios.
- Formulario para crear una cita (nombre, teléfono, día y hora).
- Almacenamiento de citas en SQLite.
- Visualización de todas las citas programadas.
